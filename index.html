OpenBuilds CONTROL - Machine Driver

OpenBuilds Machine Driver is an application for connecting to, and controlling, your CNC machine.

This will allow you to

  • Interface with, and Jog your machine
  • Run GCODE Jobs
  • Set Zero coordinates
  • Integrate with cam.openbuilds.com
  • and even help with your Firmware configuration
  • Windows
  • Mac
  • Linux

OpenBuilds CAM - GCODE Generator

OpenBuilds CAM is a web-based application for converting SVG, DXF and Bitmap drawings, to GCODE

This will allow you to

  • Import DXF/SVG/PNG/BMP/JPG files
  • Import from a built-in Parts Library
  • Setup toolpaths for CNC Routing, Laser cutting and Plasma Cutting
  • Generate GCODE
  • Send GCODE to the OpenBuilds Machine Library
Browse to https://cam.openbuilds.com
